# Juego 2D Unity

Proyecto final desarrollado en Unity 2D.

## Descripción
Es un videojuego sencillo donde el jugador debe moverse por el escenario, recolectar monedas y evitar enemigos.

## Características
- Movimiento del personaje
- Recolección de monedas
- Sistema de puntos
- Enemigos con colisión
- Pantalla de Game Over
- Reinicio del juego

## Tecnologías utilizadas
- Unity
- C#
- Visual Studio

## Cómo ejecutar
1. Descargar el proyecto o ejecutable.
2. Abrir archivo `.exe`
3. Jugar
